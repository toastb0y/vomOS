# vomOS
project repo for use in the future, just setting it up but it's not in use rn.

for now, you can view the files in the project. some of them may not have anything inside, but they're here to show the way i plan to organize this project. as it says in the description, I want to design vomOS do be a modular toy OS that allows you to easily make your own apps for it*. it sounds ambitious, and it really is, at least for me. I've only recently started coding for real in Lua, so it may take a while until i gain the skills neccesary to develop this thing...
so until then, if you plan on making apps for vomOS, or just want to see what its all about, you can study the files here and... use your imagination lol
or maybe you can use this to try and make vomOS or something similar yourself.
either way, I hope you wont mind waiting a bit, haha

*every app should be kept in it's own folder inside the "apps" folder, located inside "m_system" with it's own lua file(s) and it's art assets kept in the "2dAssets" folder.
